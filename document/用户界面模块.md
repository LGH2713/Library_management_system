## 用户界面模块

**隶属角色：** 普通用户

### 功能

1. 预约借书
2. 还书
3. 查询个人信息
4. 查询图书信息
5. 查看公告

### 用户信息表设计

**表名：** user

| 字段名     | 类型     | 约束        | 注释    |
| ---------- | -------- | ----------- | ------- |
| u_id       | int      | primary key | 用户 ID |
| u_name     | char(20) | not null    | 姓名    |
| u_sex      | char(2)  | default(男) | 性别    |
| u_password | char(20) | not null    | 密码    |
| u_mail     | char(20) | not null    | 邮箱    |
