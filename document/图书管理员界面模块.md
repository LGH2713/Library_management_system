## 图书管理员界面模块

**隶属角色：** 一般管理员

### 功能

1. 添加图书
2. 删除图书
3. 异常还书处罚
4. 发布公告
5. 个人信息

### 图书管理员个人信息表设计

| 字段名   | 类型     | 约束        | 注释          |
| -------- | -------- | ----------- | ------------- |
| id       | char(10) | primary key | 图书管理员 ID |
| name     | char(20) | not null    | 姓名          |
| mail     | char(20) | not null    | 邮箱          |
| password | char(20) | not null    | 密码          |
