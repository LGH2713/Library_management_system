## 公告模块

### 公告信息表设计

| 字段名       | 类型      | 约束              | 注释          |
| ------------ | --------- | ----------------- | ------------- |
| a_id         | char(10)  | primary key       | 公告 ID       |
| a_time       | datetime  | not null          | 公告时间      |
| a_p_id       | char(10)  | foreign key       | 公告发布者 id |
| announcement | char(250) | default(暂无公告) | 公告内容      |
